                How LaunchForge works
                
            
    
            
    
                
    What it does
    
                
    LaunchForge builds a launch sequence: a list of things to do in order, such as open a program, wait a few seconds, then start a game. It turns that list into a .bat (or .ps1) script you download and double-click to run. Nothing runs from this page and nothing is uploaded; the script runs on your own PC.
    

                
    Quick start
    
                
      
                    
    1. Click + Add and choose a step, such as Launch App or Launch Steam Game.
      2. 
                    
    2. Fill in its details on the right. Hover any ? for guidance.
      3. 
                    
    3. Add more steps and reorder them with the ▲ ▼ arrows. They run top to bottom.
      4. 
                    
    4. Watch the Preview update as you go.
      5. 
                    
    5. Click Export .bat, then double-click the downloaded file to run your sequence.
      6. 
                
    
                
    Load an example sequence to see how it fits together.
    

                
    The steps
    
                
      
                    
    • APP Launch App: open a program or shortcut.
      • 
                    
    • STEAM Launch Steam Game: start a game by its App ID.
      • 
                    
    • WAIT Wait: pause for a number of seconds.
      • 
                    
    • KILL Kill Process: force-close a program.
      • 
                    
    • WAIT+ Wait for Process Start: pause until a program is running.
      • 
                    
    • WAIT- Wait for Process Exit: pause until a program closes.
      • 
                    
    • PS Run PowerShell: advanced, run your own commands.
      • 
                
    

                
    Adding a program
    
                
    For Launch App, the quickest way to get the path is to find the program in your Start menu or on your desktop, right-click it, choose Copy as path, and paste it into the path box. LaunchForge strips the surrounding quotes for you.
    

                
    Steam games
    
                
    Use Pick game to search the full Steam catalogue by name, or paste an App ID. Launching by App ID keeps working even if you later move or delete a desktop shortcut.
    

                
    Which file: .bat or .ps1?
    
                
    Both produce the same launch sequence. The difference is how you run them:
    
                
      
                    
    • .bat (recommended for most people): double-click and it runs, no setup. Best for straightforward launches and waits, and for sharing with others.
      • 
                    
    • .ps1 (PowerShell): better when a sequence includes a Run PowerShell step, or when a launched program is noisy (some game launchers flood the window or hold it open under .bat). The catch: Windows will not run a .ps1 on double-click, and may block it. See the next section.
      • 
                
    
                
    Rule of thumb: start with .bat. Reach for .ps1 only if you have a Run PowerShell step or a launcher misbehaves under .bat.
    

                
    Running a .ps1 (and making a shortcut for it)
    
                
    Double-clicking a .ps1 opens it in Notepad rather than running it, and Windows' execution policy can block it. To run it on demand, make a shortcut that calls PowerShell directly:
    
                
      
                    
    1. Save your exported .ps1 somewhere it will stay put.
      2. 
                    
    2. Paste that folder's path below to get a ready-to-use shortcut target.
      3. 
                    
    3. Right-click your Desktop, choose New → Shortcut, paste the target, and give it a name.
      4. 
                
    
                
    
                    
                    
                    
                    
    
                        
                        
                    
    
                    
    Uses the current sequence name. If you move the script later, redo this to refresh the target.
    
                
    

                
    Scripts can open, close, and interact with programs on your PC. Only run scripts you built yourself or trust the source of, and have a quick read of the Preview before running.
    

                
    Saving and reopening
    
                
    Save .lfjson keeps your sequence as a project file. Import loads one back, and it can also read an existing .bat or .ps1 so you can pick up where you left off.

                Before you run this
                
            
    
            
    
                
    Your antivirus or Windows SmartScreen may warn about this script. That is expected, and it does not mean something is wrong.
    
                
    A launch script starts programs, and can close them or run PowerShell. That behaviour resembles what some malware does, so security tools sometimes flag any script that does it, even a harmless one. This is a false positive.
    
                
    It is safe to run because:
    
                
      
                    
    • You built this sequence yourself.
      • 
                    
    • The script is plain text you can read in full. Check the Preview before running it.
      • 
                    
    • Nothing is hidden, and nothing is sent anywhere.
      • 
                
    
                
    If a warning appears: on SmartScreen click More info then Run anyway; in your antivirus, choose to allow or whitelist the file.
