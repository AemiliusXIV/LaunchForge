                How LaunchForge works
                
            
    
            
    
                
    What it does
    
                
    LaunchForge builds a launch sequence: a list of things to do in order, such as open a program, wait a few seconds, then start a game. It turns that list into a .bat (or .ps1) script you download and double-click to run. Nothing runs from this page and nothing is uploaded; the script runs on your own PC.
    

                
    Quick start
    
                
      
                    
    1. Click + Add and choose a step, such as Launch App or Launch Steam Game.
      2. 
                    
    2. Fill in its details on the right. Hover any ? for guidance.
      3. 
                    
    3. Add more steps and reorder them with the ▲ ▼ arrows. They run top to bottom.
      4. 
                    
    4. Watch the Preview update as you go.
      5. 
                    
    5. Click Export .bat, then double-click the downloaded file to run your sequence.
      6. 
                
    
                
    Load an example sequence to see how it fits together.
    

                
    The steps
    
                
      
                    
    • APP Launch App: open a program or shortcut.
      • 
                    
    • STEAM Launch Steam Game: start a game by its App ID.
      • 
                    
    • WAIT Wait: pause for a number of seconds.
      • 
                    
    • KILL Kill Process: force-close a program.
      • 
                    
    • WAIT+ Wait for Process Start: pause until a program is running.
      • 
                    
    • WAIT- Wait for Process Exit: pause until a program closes.
      • 
                    
    • PS Run PowerShell: advanced, run your own commands.
      • 
                
    

                
    Adding a program
    
                
    For Launch App, the quickest way to get the path is to find the program in your Start menu or on your desktop, right-click it, choose Copy as path, and paste it into the path box. LaunchForge strips the surrounding quotes for you.
    

                
    Steam games
    
                
    Use Pick game to search the full Steam catalogue by name, or paste an App ID. Launching by App ID keeps working even if you later move or delete a desktop shortcut.
    

                
    Scripts can open, close, and interact with programs on your PC. Only run scripts you built yourself or trust the source of, and have a quick read of the Preview before running.
    

                
    Saving and reopening
    
                
    Save .lfjson keeps your sequence as a project file. Import loads one back, and it can also read an existing .bat or .ps1 so you can pick up where you left off.
